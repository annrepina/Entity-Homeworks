Создать базу данных и ее схему используя миграции EF

- Создать DbContext и добавить в него таблицу Orders
- Соответственно необходимо создать сущность Order которая будет содержать:
	- поле Id в качестве первичного ключа (Not null)
	- поле Summ для хранения суммы (Not null)
- Создать инициализирующую миграцию для этой схемы
- Заполнить некоторыми значениями по умолчанию (можно прям в первой миграции если разберетесь)
- Добавить в сущность Order поле Date допускающее значение NULL
- Создать миграцию с новым полем
- Изменить поле Date на NOT NULL и заполнить столбец значениями по умолчанию при помощи миграции
- Удалить поле Date и создать соответствующую миргацию

Необходимо обратить внимание что для каждой миграции должна работать как прямая, так и обратная миграция
